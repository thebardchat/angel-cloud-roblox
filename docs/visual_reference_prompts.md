## STYLE GUIDE FOR ALL PROMPTS

**Always include these modifiers for consistency:**
- "Roblox game style" or "Roblox art style"
- "Cinematic lighting" for dramatic shots
- "Golden hour" or "god rays" for warm atmosphere
- "Floating cloud islands" for environment context
- "16:9 widescreen" for key art
- "Mobile-friendly UI" for HUD shots

**Color palette per layer:**
- Nursery: Warm gold, cream, soft white
- Meadow: Cyan, sky blue, white
- Canopy: Emerald green, gold dappled light
- Stormwall: Electric purple, dark clouds, cyan lightning
- Luminance: Brilliant white, crystal, lens flare
- Empyrean: Prismatic aurora, shifting colors, gold + all colors

**Never include:**
- Violence, weapons, blood
- Scary or dark horror elements
- Real brand logos
- Text that isn't game UI
